# Software Development 1 Learning Path

## Primary Goal

Learn the frontend basics needed to contribute small, focused improvements to the Blaze Diagnostics app.

## Study Focus

- GitHub basics
- HTML, CSS, and JavaScript
- TypeScript basics
- React components and props
- Next.js page structure
- Tailwind CSS utility classes
- Reading existing code
- Small UI updates

## Start With

1. `docs/training/GITHUB_WORKFLOW.md`
2. `frontend/src/app/page.tsx`
3. `frontend/src/features/dashboard/`
4. `frontend/src/features/customers/`
5. `frontend/src/features/vehicles/`

## Suitable Starter Tasks

- Update text labels or empty states.
- Create a simple status badge component.
- Improve spacing and layout using Tailwind.
- Build a simple static page section.
- Document what a component does.
- Fix a small display bug.
