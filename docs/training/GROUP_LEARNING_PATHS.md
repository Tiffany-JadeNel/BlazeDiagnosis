# Group Learning Paths

Use this document to find the correct training path for your group.

## Current Internship Groups

- Software Engineering 1
- Software Engineering 2
- Software Development 1
- Software Development 2
- Cloud Administration
- Cyber Security

Architectural Draughting students use the separate Synergy Inc. Architectural Draughting repository.

## Recommended Focus by Group

| Group | Main Focus | Suitable First Tasks |
|---|---|---|
| Software Engineering 1 | System understanding, entities, workflows, API/database concepts | User story review, entity mapping, workflow notes |
| Software Engineering 2 | Architecture, backend structure, database modelling, API planning | Schema review, backend service planning, role access design |
| Software Development 1 | Frontend basics, components, Tailwind, simple UI tasks | Small component updates, simple pages, styling tasks |
| Software Development 2 | Feature implementation, API integration, validation, testing | CRUD screens, API wrappers, form validation, bug fixes |
| Cloud Administration | Environment setup, deployment, CI/CD, backups, monitoring | Setup guide, env docs, CI review, deployment checklist |
| Cyber Security | OWASP, access control, secret hygiene, customer data protection | Security checklist, risk review, RBAC test notes |
