# Software Development 2 Learning Path

## Primary Goal

Work on more complete frontend features and integrations using Next.js, TypeScript, Tailwind CSS, and API wrappers.

## Study Focus

- Next.js routing and project structure
- React components and state
- TypeScript types and interfaces
- API client usage
- Form handling and validation
- CRUD screens
- Feature folder structure
- Manual testing and bug reporting

## Start With

1. `frontend/src/lib/api-client.ts`
2. `frontend/src/services/endpoints.ts`
3. `frontend/src/features/jobs/`
4. `frontend/src/features/quotes/`
5. `frontend/src/features/customers/`

## Suitable Starter Tasks

- Build or improve a customer list page.
- Build a vehicle details screen.
- Add a job status display section.
- Wire a component to an existing API wrapper.
- Add validation notes for a form.
- Create a manual test checklist for a feature.
